3️⃣  INSTALAR BACKEND
    └─ cd backend && npm install && npm run dev

    └─ cd frontend && npm install && npm run dev

5️⃣  ACESSAR
    └─ http://localhost:3000

┌──────────────────────────────────────────────────────────────────────────┐
│  💾 BANCO DE DADOS                                                        │
└──────────────────────────────────────────────────────────────────────────┘

Tabela: equipamentos
├── id (Auto-increment)
├── id_interno (VARCHAR, UNIQUE)
├── tipo (VARCHAR)
├── marca (VARCHAR)
├── modelo (VARCHAR)
├── numero_serie (VARCHAR)
├── status_integridade (ENUM)
├── pecas_faltantes (TEXT)
├── status_limpeza (ENUM)
├── testado (BOOLEAN)
├── data_registro (TIMESTAMP)
└── observacoes (TEXT)

Índices: tipo, status_integridade, data_registro

Dados de Exemplo:
1. Monitor Samsung (Completo)
2. Notebook Dell (Incompleto - Falta RAM)
3. Monitor LG (Completo)
4. Desktop HP (Incompleto - Falta Processador)
